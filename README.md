# 🤖 AI FAQ Chatbot

An intelligent FAQ Chatbot developed as part of the **CodeAlpha Artificial Intelligence Internship**. This project uses Natural Language Processing (NLP) techniques to understand user queries and provide the most relevant answers.

- Interactive Chat Interface
- NLP-Based Question Processing
- TF-IDF Vectorization
- Cosine Similarity Matching
- Intelligent Response Generation
- User-Friendly Interface
- Offline Functionality

- Fast and Lightweight

## 🧠 Working Principle

1. User enters a question.
2. Questions are converted into numerical vectors using TF-IDF Vectorization.
3. Cosine Similarity is used to find the most relevant FAQ.
4. The chatbot returns the best matching answer.

## 📷 Output

The chatbot provides intelligent responses based on the similarity between the user's query and predefined FAQs.
